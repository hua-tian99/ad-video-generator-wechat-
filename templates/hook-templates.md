# 钩子设计模板

## 微信视频号专用钩子模板

### 第1层：视觉钩子（0-1秒）

**目标**：用真实、接地气的视觉元素立即抓住用户注意力

**微信视频号特点**：
- 竖屏拍摄（9:16格式）
- 手机拍摄感（轻微晃动）
- 真实场景（不是摄影棚）
- 大字幕立即出现

**模板选项**：

#### 模板A：真实场景手持展示
```
竖屏画面，手机拍摄感，轻微晃动。
一只手自然地拿着[产品名称]，产品位于画面中央，
自然光线（窗户光或室内灯光），真实家居背景。
同时大字幕立即出现在画面中部偏上："[痛点文案]"
（超大字号、白色字+黑色描边）
```

#### 模板B：痛点场景特写
```
竖屏画面，手机拍摄感。
画面聚焦在[痛点场景]，例如：
- 捂着耳朵听不清的表情（助听器）
- 捂着腰疼痛的动作（按摩仪）
- 失眠翻来覆去的场景（助眠产品）
大字幕立即出现："[口语化痛点]"
（超大字号、白色字+黑色描边、画面中部偏上）
```

#### 模板C：产品使用场景特写
```
竖屏画面，真实生活场景。
[人物]正在[真实场景]中使用产品，
素人感、日常服装、自然光线。
大字幕立即出现："[痛点或效果文案]"
```

---

### 第2层：痛点钩子（1-2秒）

**目标**：引发目标用户的情感共鸣，让用户意识到"这说的就是我"

**微信视频号特点**：
- 口语化表达（说人话，不说广告词）
- 超大字幕（确保无声也能看懂）
- 情感共鸣词（真的、太、好、很等）

**文案公式**：
```
[痛点描述] + [情感强化词]
```

**示例（口语化表达）**：
- "听不清别人说话，真的太尴尬了" （助听器）
- "腰疼到直不起来，太难受了" （按摩仪/护腰）
- "每天失眠到天亮，快崩溃了" （助眠产品）
- "血压忽高忽低，心里总是慌" （血压计）
- "关节疼得走不动路，太痛苦了" （护膝产品）
- "眼睛干涩得睁不开，好难受" （眼药水/护眼产品）

**展示方式**：
- 位置：画面中部偏上（避开底部1/5区域）
- 字体：超大字号、加粗、白色字+黑色描边
- 动画：逐字出现或整句弹出
- 时长：1-2秒

---

### 第3层：解决方案钩子（2-3秒）

**目标**：自然过渡，口碑式表达，建立"问题-解决方案"的心理连接

**微信视频号特点**：
- 口碑式表达（像朋友推荐）
- 自然过渡（不是硬广）
- 大字幕持续显示

**文案公式**：
```
"直到我[动作] + [产品名称] + [效果]"
或
"用了[产品名称] + [效果]"
```

**示例（口碑式表达）**：
- "直到我用了XX，真的改变了" （通用）
- "用了XX之后，终于好了" （通用）
- "XX真的有用，不骗你" （强调真实）
- "朋友推荐的XX，真没让我失望" （口碑）
- "试了XX，没想到这么好用" （惊喜）

**展示方式**：
- 位置：画面中部偏上（接续痛点文案）
- 字体：超大字号、品牌名可加粗或变色
- 动画：痛点文案淡出后立即出现，或直接替换
- 时长：1-2秒

---

## 三层钩子机制（0-3秒）

### 第1层：视觉钩子（0-1秒）

**目标**：用强烈的视觉元素立即抓住用户注意力

**模板选项**：

#### 模板A：产品特写手持展示
```
画面从模糊逐渐清晰，一只手优雅地托举着[产品名称]，产品位于画面中央，
柔和的光线从侧面照射，突出产品的[视觉特点：小巧/科技感/专业感]。
镜头缓慢拉近，聚焦产品细节。
```

#### 模板B：产品动态展示
```
[产品名称]在纯色背景前缓慢旋转，光线随着旋转角度变化，
产品表面反射出柔和的光泽，突出其[材质特点]和[设计美感]。
```

#### 模板C：产品使用场景特写
```
画面聚焦在[使用场景]中的产品，[人物动作]正在[使用产品]，
镜头从侧面特写产品与[身体部位]的接触，展现其[舒适性/便捷性]。
```

---

### 第2层：痛点钩子（1-2秒）

**目标**：引发目标用户的情感共鸣，让用户意识到"这说的就是我"

**文案公式**：
```
[目标人群] + [痛点描述] + [情感词]
```

**示例**：
- "家人们听不清太闹心" （助听器）
- "腰酸背痛太难受了" （按摩仪）
- "睡眠质量差很困扰" （助眠产品）
- "血压不稳心里慌" （血压计）
- "关节疼痛走不动" （护膝产品）

**展示方式**：
- 位置：画面下方1/3处
- 字体：清晰易读，白色或黄色
- 动画：从下方滑入或淡入
- 时长：1-2秒

---

### 第3层：解决方案钩子（2-3秒）

**目标**：立即给出产品定位，建立"问题-解决方案"的心理连接

**文案公式**：
```
[行动词] + [产品名称] + [效果承诺]
```

**示例**：
- "别愁 [品牌名][产品名]封神" （强势定位）
- "[产品名]一招搞定" （简单直接）
- "[产品名]让你告别[痛点]" （解决方案）
- "有了[产品名] [痛点]不再是问题" （承诺式）

**展示方式**：
- 位置：画面下方1/3处（接续痛点文案）
- 字体：比痛点文案稍大，强调品牌名
- 动画：痛点文案淡出后立即出现
- 时长：1-2秒

## 完整钩子设计示例

### 微信视频号示例1：助听器产品

**视觉钩子（0-1秒）**：
```
竖屏画面（9:16），手机拍摄感，轻微晃动。
真实家居场景，自然光线。
一只手自然地拿着小巧的助听器，产品位于画面中央。
大字幕立即出现："听不清别人说话，真的太尴尬了"
（超大字号、白色字+黑色描边、画面中部偏上）
```

**痛点钩子（1-2秒）**：
```
大字幕持续显示："听不清别人说话，真的太尴尬了"
画面可切换到人物捂耳朵、皱眉的真实表情
位置：画面中部偏上，超大字号，白色字+黑色描边
```

**解决方案钩子（2-3秒）**：
```
大字幕切换为："直到我用了自然之声，真的改变了"
画面切换到人物微笑、自然交流的场景
品牌名"自然之声"可加粗或变色强调
```

---

### 微信视频号示例2：按摩仪产品

**视觉钩子（0-1秒）**：
```
竖屏画面，手机拍摄感。
画面聚焦在人物捂着腰、表情痛苦的场景。
真实办公室或家居环境，自然光线。
大字幕立即出现："腰疼到直不起来，太难受了"
（超大字号、白色字+黑色描边）
```

**痛点钩子（1-2秒）**：
```
大字幕持续显示："腰疼到直不起来，太难受了"
画面展示人物尝试直腰但很困难的动作
位置：画面中部偏上
```

**解决方案钩子（2-3秒）**：
```
大字幕切换为："用了XX按摩仪，终于好了"
画面切换到人物使用按摩仪、表情舒适的场景
品牌名加粗强调
```

## 钩子设计检查清单

### 微信视频号专用检查清单

设计完钩子后，检查以下要点：

- [ ] 是否竖屏9:16格式？
- [ ] 是否有手机拍摄感（轻微晃动）？
- [ ] 是否使用真实场景（不是摄影棚）？
- [ ] 大字幕是否立即出现（0-1秒内）？
- [ ] 字幕是否够大（超大字号）？
- [ ] 字幕是否高对比度（白色字+黑色描边）？
- [ ] 字幕位置是否避开UI遮挡区（画面中部偏上）？
- [ ] 痛点描述是否口语化（说人话）？
- [ ] 痛点是否引发情感共鸣（有情感强化词）？
- [ ] 解决方案是否口碑式表达（像朋友推荐）？
- [ ] 三层钩子是否在3秒内完成？
- [ ] 无声观看是否能理解（文字化）？
- [ ] 视觉和文案是否协调配合？

### 传统广告检查清单（保留作为对比）

设计完钩子后，检查以下要点：

- [ ] 视觉钩子是否在1秒内抓住眼球？
- [ ] 痛点钩子是否精准描述目标用户的痛点？
- [ ] 痛点描述是否引发情感共鸣？
- [ ] 解决方案钩子是否立即给出产品定位？
- [ ] 三层钩子是否在3秒内完成？
- [ ] 文案是否简洁有力（每句不超过10个字）？
- [ ] 视觉和文案是否协调配合？
