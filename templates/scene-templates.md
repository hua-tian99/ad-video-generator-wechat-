# 案例场景模板

## 微信视频号场景展示方式

### 快速切换式（15-30秒视频）

适用于短视频，快速展示多个场景。

**展示方式**：
- 每个场景1-2秒
- 大字幕显示场景名称
- 快速跳切，不拖沓

**示例**：
```
画面快速切换：
- 1秒：居家场景 + 大字幕"居家休息"
- 1秒：办公场景 + 大字幕"办公室工作"
- 1秒：户外场景 + 大字幕"出门遛弯"
```

### 单场景深入式（30-60秒视频）

适用于需要深入展示的场景。

**展示方式**：
- 1-2个场景
- 每个场景5-10秒
- 展示使用过程和效果

**示例**：
```
场景1（5秒）：居家使用
- 画面：真实家居环境，人物使用产品
- 大字幕："在家也能轻松用"

场景2（5秒）：户外使用
- 画面：户外环境，人物使用产品
- 大字幕："出门也不怕"
```

### 对比展示式

适用于需要展示效果的产品。

**展示方式**：
- 使用前 vs 使用后
- 左右分屏或上下分屏（竖屏适配）
- 大字幕标注"使用前" "使用后"

**示例**：
```
画面分为上下两部分（竖屏）：
- 上半部分：使用前（痛苦表情）+ 大字幕"使用前"
- 下半部分：使用后（舒适表情）+ 大字幕"使用后"
```

## 竖屏场景拍摄要点

### 构图原则
- **人物居中**：主体在画面中央
- **上下留白**：避开顶部和底部UI遮挡区域
- **景深适中**：不过度虚化背景

### 真实感营造
- **自然光线**：窗户光、室内灯光，不用专业打光
- **生活化场景**：真实家居、办公室、户外，不是摄影棚
- **随意感**：不刻意摆拍，自然状态
- **手机拍摄感**：轻微晃动，增加真实感

---

## 场景具体化原则

❌ **避免使用的泛泛描述**：
- "日常使用"
- "各种场景"
- "多种情况"
- "随时随地"

✅ **应该使用的具体场景**：
- "居家休息"
- "陪娃读绘本"
- "出门遛弯"
- "办公室工作"
- "户外运动"

---

## 场景设计框架

### 1. 场景数量
- 标准配置：3-5个场景
- 不贪多，确保每个场景都有画面或文案支撑

### 2. 场景覆盖原则
- 覆盖目标用户的主要生活场景
- 场景之间有差异性（室内/户外、静态/动态、独处/社交）
- 每个场景都能让用户产生代入感

### 3. 场景展示方式
- **文案展示**：在画面上直接显示场景文案（如"居家嘴嘘"）
- **画面展示**：通过视觉画面展现场景（如人物在家中使用产品）
- **组合展示**：文案+画面同时出现

## 场景文案设计技巧

### 1. 使用动词+名词结构
- "陪娃读绘本"（动词+名词）
- "出门遛弯"（动词+名词）
- "办公室久坐"（名词+动词）

### 2. 突出场景特点
- 时间特点："晨起监测"、"睡前理疗"
- 地点特点："居家嘴嘘"、"办公室工作"
- 活动特点："运动后恢复"、"朋友聚会"

### 3. 保持简洁
- 每个场景文案控制在4-6个字
- 易读易记，朗朗上口

---

## 场景展示时机

在视频中的最佳展示位置：
- **时间**：通常在功能展示后（15-25秒）
- **方式**：快速切换展示多个场景，或单个场景配合画面
- **时长**：每个场景展示2-3秒

---

## 场景设计检查清单

设计完场景后，检查以下要点：

- [ ] 场景是否具体、可视化？
- [ ] 场景数量是否合适（3-5个）？
- [ ] 场景是否覆盖目标用户的主要生活场景？
- [ ] 场景之间是否有差异性？
- [ ] 场景文案是否简洁（4-6个字）？
- [ ] 场景是否能让用户产生代入感？
- [ ] 场景展示是否与产品功能相关？
