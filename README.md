# WeChat Video Account Ad Prompt Generator

> Claude Code skill for generating video prompts for Seendance 2.0 AI video generation model. Creates short-form vertical video (15-60s) prompts optimized for WeChat Video Accounts, targeting healthcare and medical products.

**Key Features:**
- Claude Code skill with interactive Q&A workflow
- Generates prompts for Seendance 2.0 video model via 即梦 (Jimeng) platform
- 9:16 vertical format optimization for mobile viewing
- 3-layer hook mechanism for first 3 seconds
- Healthcare and medical products only

# 微信视频号短视频广告脚本生成器

## 项目简介

这是一个 **Claude Code 技能**，专为保健品和医疗产品设计，用于生成适合微信视频号平台的竖屏短视频（15-60秒）视频提示词。生成的提示词可直接在 **即梦平台** 使用 **Seendance 2.0 视频生成模型** 生成实际视频。

**重要说明**：本技能仅适用于保健品和医疗产品。

通过5轮交互式问答收集产品信息，自动生成专业的视频提示词，无需手动剪辑视频。

## 技术栈

- **Claude Code 技能系统**：交互式问答和提示词生成
- **Seendance 2.0**：AI视频生成模型
- **即梦平台**：视频生成服务平台

## 核心特性

### 🎯 微信视频号优化
- **竖屏9:16格式**：严格遵守1080x1920竖屏比例，适配移动端观看
- **UI安全区域**：顶部和底部各留1/5安全区域，避开平台UI遮挡
- **原生内容风格**：手机拍摄感、素人感、真实场景，融入用户信息流

### ⚡ 三层钩子机制（0-3秒）
- **视觉钩子**（0-1秒）：产品特写、手持展示、真实场景
- **痛点钩子**（1-2秒）：口语化表达，超大字幕显示
- **解决方案钩子**（2-3秒）：自然过渡，口碑式表达

### 📱 快节奏剪辑
- 每个镜头2-5秒，快速切换保持注意力
- 跳切为主，不过度追求流畅过渡
- 前3秒关键，决定用户是否继续观看

### 📝 超大字幕设计
- 超大字号，确保小屏幕清晰阅读
- 白色字+黑色描边，高对比度
- 画面中部偏上位置，支持无声观看

### 🎬 智能脚本生成
- 交互式5轮问答收集产品信息
- 自动生成1-3个15秒视频片段
- 场景具体化，避免泛泛描述
- 持续视觉元素贯穿全程

## 使用前提

本项目是一个 **Claude Code 技能**，需要以下环境：

- **Claude Code CLI**：Anthropic 官方命令行工具
- **即梦平台账号**：用于使用 Seendance 2.0 模型生成视频
- **使用方式**：在 Claude Code 中加载本技能（通过 skill.md 配置文件）

## 快速开始

### 1. 在 Claude Code 中加载技能

在 Claude Code CLI 中加载本技能（技能会自动读取 skill.md 配置）。

### 2. 启动交互式问答

AI助手将引导你完成5轮问答：

**第1轮：基本信息**
- 产品名称、类别、品牌
- 视频时长偏好（15-30秒/30-45秒/45-60秒）
- 内容风格偏好（知识科普/场景种草/痛点共鸣/对比测评）

**第2轮：钩子设计要素**
- 核心痛点（口语化表达）
- 产品定位（接地气表达）
- 视觉特点

**第3轮：功能与场景**
- 核心功能（3-5个）
- 技术特点
- 具体使用场景（必须具体）
- 产品优势

**第4轮：营销与持续视觉元素**
- 促销活动
- 持续宣传语
- 宣传语显示位置
- 价格定位

**第5轮：视觉素材**
- 产品外观特征
- 产品包装特征
- 使用场景图片描述

### 3. 获取生成的视频提示词

Claude Code 将生成包含以下内容的完整提示词：
- 产品信息摘要
- 视频整体规划
- 每个片段的详细提示词（画面、镜头、文字、持续视觉元素等）
- 整体风格说明
- 使用建议

### 4. 在即梦平台生成视频

将生成的提示词复制到 **即梦平台**，选择 **Seendance 2.0 模型**，即可自动生成实际视频，无需手动剪辑。

## 输出格式示例

生成的脚本包含1-3个15秒片段，每个片段包括：

```
### 片段 1 (0-15秒)

**钩子设计**：
- 视觉钩子（0-1秒）：[产品特写，手机拍摄感]
- 痛点钩子（1-2秒）：大字幕"[口语化痛点]"
- 解决方案钩子（2-3秒）：大字幕"[口碑式解决方案]"

**画面描述**：
[详细的竖屏9:16画面描述，真实感、素人感]

**镜头运动**：手持感、快速切换、跳切

**文字内容**：
- 1-2秒：大字幕"[文案]"（超大字号、白色字+黑色描边、画面中部偏上）

**持续视觉元素**：
- 位置：顶部/底部
- 内容："[宣传语]"
- 显示时长：贯穿整个片段

**风格标签**：真实接地气、快节奏、竖屏9:16、1080P
```

## 与电视版的区别

| 特性 | 短视频版（本项目） | 电视版 |
|------|------------------|--------|
| **格式** | 竖屏9:16（1080x1920） | 横屏16:9（4K） |
| **平台** | 微信视频号 | 电视、专业媒体 |
| **风格** | 手机拍摄感、素人感、真实场景 | 专业摄影、演员、摄影棚 |
| **剪辑** | 快节奏、跳切、每镜头2-5秒 | 流畅过渡、专业转场 |
| **字幕** | 超大字幕、支持无声观看 | 标准字幕 |
| **片段** | 1-3个15秒片段 | 2-5个15秒片段 |
| **目标** | 快速种草、社交互动 | 深度展示、品牌建设 |

## 模板系统说明

项目包含完整的模板库（`templates/`目录），涵盖：

- **钩子设计模板**：视觉钩子、痛点钩子、解决方案钩子的设计范式
- **场景设计模板**：具体化场景描述的方法和案例
- **持续视觉元素模板**：宣传语、logo、促销信息的展示方式
- **片段结构模板**：不同类型片段的标准结构
- **过渡设计模板**：片段间的快速跳切设计
- **视觉风格指南**：真实感、素人感、手机拍摄感的实现方法
